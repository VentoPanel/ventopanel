## v0.1.8

### Highlights

- Full CRUD UI for servers and sites — create, edit, delete, with confirmation dialogs.
- Action buttons in each table row: **Connect** and **Provision** for servers; **Deploy** for sites.
- Toast notifications (Sonner) with success/error feedback for all mutations.
- New Shadcn UI primitives: Dialog, Input, Label.
- TanStack Query mutations with automatic cache invalidation after every write.
