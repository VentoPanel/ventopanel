## v0.1.11 — Audit Log UI

### Highlights

- New **/audit** page: full audit trail for all server and site status transitions.
- Color-coded status badges (red = error/denied, green = success, blue = in-progress).
- Filter by resource type: All / Server / Site.
- Cursor-based pagination — "Load more" button fetches the next 50 events without re-fetching previous pages.
- Direct link in sidebar navigation.
