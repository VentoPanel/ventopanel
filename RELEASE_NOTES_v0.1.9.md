## v0.1.9

### Highlights

- Auto-refresh for servers and sites every **15 seconds** (paused when tab is in background).
- `RefreshIndicator` component with:
  - Green dot → data is fresh
  - Spinner → request in flight
  - "Updated Xs ago" counter (updates every second)
  - Manual refresh button (⟳)
- Indicators added to Dashboard, Servers page, and Sites page.
